利用Hexo+GitHub搭建静态博客
本地保留Markdown文件，可随时生成一个博客

第一次接触Hexo是在研究生的时候，2016年过年的时候简单搞了一下，配置了Next主题，觉得美美的。然而后面并没有写太多有用的东西在上面，直到这次查资料看到了Maupassant主题，一下子就被吸引了，决定在“繁忙”的工作之余，再次倒腾一次。
可能因为之前弄过的缘故，这次一切都比较顺利，一个晚上基本搞定。
